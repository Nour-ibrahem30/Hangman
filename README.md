# Hangman Game

A classic Hangman game where you guess the word before the full hangman figure is drawn.  
Built with HTML, CSS, and JavaScript, and improved with **Vibe Coding** for better code quality and organization.

---

## Project Description

This is an interactive word guessing game with categories. Incorrect guesses gradually reveal parts of the hangman figure.  
The interface is clean, responsive, and user-friendly for an enjoyable experience.

---

## How to Run

1. Download or clone the repository.  
2. Open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari).  
3. Start playing and have fun!

---

## Features

- Interactive and intuitive UI.  
- Responsive design for different screen sizes.  
- Sound effects for success and failure feedback.  
- Hangman figure parts show progressively on wrong guesses.  
- Code improved and organized using Vibe Coding methodology.

---

## Requirements

- Modern browser with support for HTML5, CSS3, and ES6 JavaScript.
